1. Invalid GTIN check digit

A mistyped barcode fails Google's check-digit test. Fix: correct it, or set identifier_exists = no — never invent one. Full guide.

2. No identifier and identifier_exists not set

Own-brand items with no GTIN/MPN still need identifier_exists = no. Fix: add that field for unbranded products.

3. Price format / currency

Google wants 24.99 GBP, not £24.99 or 24,99. Fix: number + ISO currency code, dot decimal.

4. Availability value not recognised

available / in stock / yes are rejected. Fix: use in_stock, out_of_stock, preorder, or backorder.

5. Missing image_link

No image, or a broken/placeholder image URL. Fix: a valid https:// product image, no watermarks or promo text.

6. Promotional text in the title

"Free shipping", "Sale", "Best price" in titles get flagged. Fix: describe the product, don't advertise it.

7. ALL-CAPS titles

Fully capitalised titles are a classic trigger. Fix: normal title case.

8. Title over 150 characters

Long titles get truncated or penalised. Fix: trim to ≤150, front-load key words.

9. Missing description

Empty descriptions (common when Shopify body HTML is blank) disapprove items. Fix: add a real description, no promo text.

10. Missing id or link

Every product needs a unique id and a valid landing-page link. Fix: ensure both are present and unique.
